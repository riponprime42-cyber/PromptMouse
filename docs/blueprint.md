# **App Name**: PromptMuse

## Core Features:

- AI Prompt Generation: Generate unique and detailed text prompts for image or video creation, serving as a tool for creative inspiration based on user specifications.
- Custom Input Parameters: Allow users to provide initial guidance such as subject, style, mood, and artistic references to direct the AI's prompt generation.
- Prompt History: Maintain a history of all generated prompts, enabling users to easily revisit and reuse past ideas.
- Favorite Prompts List: Provide functionality to save and quickly access a curated list of user's preferred prompts.
- Copy to Clipboard: Implement a one-click option to instantly copy any generated or saved prompt to the clipboard for external use.
- Manual Prompt Refinement: Offer an interface for users to manually edit, expand, or fine-tune generated prompts before saving or utilizing them.

## Style Guidelines:

- Primary color: A rich, inspiring violet (#8C47D1) chosen to evoke creativity and digital artistry.
- Background color: A very dark, subtle purplish-gray (#261F2E) providing a professional and focused dark mode experience.
- Accent color: A vibrant magenta (#F075F0) that offers high contrast and adds an energetic pop, complementing the primary violet.
- Headline and body font: 'Space Grotesk' (sans-serif) for its modern, tech-inspired, and slightly scientific aesthetic, suitable for clear display of prompts.
- Utilize abstract, clean, and minimalist vector icons with subtle gradients to align with the futuristic and creative application theme.
- Adopt a clean, intuitive, and centered layout that prioritizes readability of the generated prompts and clear input fields, consistent with a high-tech studio environment.
- Incorporate subtle and smooth transitions for prompt generation, loading states, and navigation to enhance user experience without distraction, reflecting a fluid creative process.
